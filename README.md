# pia01-inteligencia-artificial
Este es un repositorio para la tarea de la unidad 1 de la asignatura **"Introducción a la Programación de Inteligencia Artificial"**
El objetivo principal es empezar a trabajar como desarrollador de IA utilizando las prácticas aprendidas en el temario:
- Crear una cuenta de **Github** y utilizarla como un posible portafolio profesional
- Crear un **primer repositorio** para exponer la tarea encomendada.
- Crear un **readme** utilizando el lenguaje de marcado **Markdown**.
- Añadir un programa "Hola Mundo" escrito en el lenguaje de programación **Python**
